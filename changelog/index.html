更新日志

[0.1.7] - 2026-04-17

新增

  • HTTP 客户端增强
    • 在 get_stock_data 方法的 period 参数支持了更多查询维度：base (基础数据), info (资讯信息), adjust (复权因子)

[0.1.6] - 2026-04-16

新增

  • HTTP 客户端增强
    • 在 HistoricalClient 的 base_url 中缺省时自动补充 http:// / https://
    • 为 HistoricalClient 增加 verify 参数，以支持关闭 SSL 证书校验（默认开启）

修复

  • 请求重试机制优化
    • 修复了因为对所有 HTTP 错误都重试导致隐藏掉真实 4xx 报错的问题，现在仅在网络错误或 500+ 服务端错误时重试

[0.1.5] - 2026-02-10

新增

  • HTTP 客户端增强
    • 新增 get_option_info 方法，支持获取期权合约基础信息
    • 支持查询上交所(SSE)和深交所(SZSE)的期权数据

[0.1.3] - 2025-11-12

新增

  • HostsManager 工具类 - 跨平台 hosts 文件管理工具
    • 支持添加/更新/删除 hosts 映射
    • 支持备份和恢复 hosts 文件
    • 支持列出和批量清除自定义条目
    • 自动权限检查和跨平台路径处理
    • 支持 Windows、Linux 和 macOS
  • 完整的 HostsManager 使用文档

修复

  • Python 3.8/3.9 兼容性问题
    • 修复 client.py 中 str | None 类型注解导致的 TypeError
    • 修复 hosts.py 中 list[str]、dict[str, str] 类型注解兼容性
    • 确保项目在 Python 3.8+ 所有版本中正常工作

[0.1.2] - 2025-11-06

新增

  • 全局配置系统，支持通过环境变量配置（前缀 HTQUANT_）
  • HSStockData 数据模型，包含完整的沪深现货行情字段
  • HTTP 客户端模块 (HistoricalClient)
    • 支持获取股票历史数据
    • 内置重试机制（最多3次）
    • 支持 OAuth 2.0 认证
  • 配置模块

变更

  • HistoricalClient 使用全局配置实例替代硬编码参数
  • connect 方法支持可选参数，默认使用配置中的用户名和密码
  • 优化导入结构，统一导出

修复

  • 修复模块无法导入的问题
  • 修正测试问题

[0.1.0] - 2025-11-06

新增

  • 初始版本发布
  • 基础项目结构
  • htQuant 模块框架
  • 基本的 HTTP 客户端实现
